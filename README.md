# Random Outfit Generator
A Tkinter app that randomly generates an outfit. Inspired by TheComeUp on YouTube.

## Technologies
* Python 3.6+
* Tkinter
* Pillow

## Local Setup
* Install all dependancies:
`pip3 install -r requirements.txt`
* Run the app.py file:
`python3 app.py`
